# Umfrage

## Beschreibung
Ein einfaches Umfrage-Projekt zum Erstellen und Verwalten von Umfragen.

## Features
- Umfragen erstellen
- Fragen hinzufügen
- Antworten sammeln
- Ergebnisse anzeigen

## Installation
```bash
# Abhängigkeiten installieren
npm install
```

## Verwendung
```bash
# Projekt starten
npm start
```

## Struktur
```
Umfrage/
├── src/
├── tests/
└── README.md
```
